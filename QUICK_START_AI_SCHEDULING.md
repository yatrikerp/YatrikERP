# Quick Start: AI Autonomous Scheduling

## 🚀 3-Step Quick Start

### 1. Start Backend
```bash
cd backend
node server.js
```
Wait for: `✅ MongoDB Connected` and `🚀 Server running on port 5000`

### 2. Start Frontend  
```bash
cd frontend
npm start
```
Opens at: `http://localhost:3000`

### 3. Use AI Scheduling
1. Login as admin: `http://localhost:3000/admin/login`
2. Go to: `http://localhost:3000/admin/autonomous-scheduling`
3. Click: **"Run Full AI Fleet Scheduling"**
4. Wait 2-5 seconds
5. View results!
